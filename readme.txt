запускаем бэк
для запуска фронта открываем в терминале директорию JS Front/Internet-store/client
И пишем команду для установки зависимостей
npm install npm uninstall  --legacy-peer-deps  --force
Далее запускаем приложение командой
npm run start

Текушие баги:
Без добавления характеристики ничего не отображается на окне девайса

Нельзя добавить больше одной характеристики

Нет мидллеера с проверкой авторизации
